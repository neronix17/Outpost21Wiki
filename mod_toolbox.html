[O21] Toolbox

The Toolbox is my core library mod, used to hold a bunch of different functionality used across many of my mods. Anyone is free to make use of the Toolbox as a dependency, or learn from the code if it helps.

Directly copying out code to be included in a mod separately can cause issues so it's best not to do that if possible. Dependency is the only supported use of the Toolbox, I won't help you export parts or with making those parts function independently.

Features

The Toolbox holds a substantial amount of functionality, and I'm not writing it all up in this custom made site so head over here to the Toolbox GitHub wiki.
